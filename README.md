chebsnake, shooting, 踩地雷

完成了踩地雷, mine.m, tiles.jpg(這個圖是上網找的)

mine(N,bn); N代表幾乘幾格，bn代表炸彈數

玩法:

    左鍵:點開格子
    
    右鍵:插旗拔旗(不能點太快，ginput()有時會讀成左鍵)
    
    滾輪:可以直接中斷遊戲
  
結束條件:

    1.點到炸彈
    
    2.非炸彈格都點完
  
內容:

    background是遊玩畫面
    
    backinfo儲存每一格的狀態
    
        -1 炸彈
        
         0 沒點過
         
         1 點過
         
         2 旗子
         
         3 炸彈旗
